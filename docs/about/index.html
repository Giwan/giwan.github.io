VOL. 23 • ISSUE 42

Hi, I'm Giwan.

I'm a senior frontend developer who likes to build. This is just one of the side projects I spend time on. My primary side project is Mytoori.com. I grep up speaking both English, at home, and Dutch, at school. Later I traveled quite a bit for work. I picked up and can get by in a few other languages. German is close enough to Dutch so I cheat a bit there. I spent a year living in France but also lost a lot of my fluency when I left. I'm currently learning and struggling with Portuguese.

I mostly develop with ReactJS but have dabbled with SvelteKit, VueJS, etc. Meta frameworks are the future and I prefer Remix or React Router.

In the backend I use NodeJS. Lately I've been using and liking Deno. Since most of the things I build are side projects, I look at services where I can host them for free. Most work with serverless functions however the ones that need AI are pushing the limits.

This site is built with Astro and TailwindCSS. I use GitHub for version control and GitHub Pages for hosting.

If you'd like to reach out, I'm moving away from twitter and instead you can find me on mastodon.social/@giwan
